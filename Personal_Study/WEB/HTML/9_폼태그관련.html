폼 관련 태그

form태그는 html에서 사용자가 입력할 수 있는 양식을 제공하는 태그이다.
form 태그 내의 input 태그들을 통해 사용자가 입력한 정보를 서버로 넘기는 역할을 한다.
* action 속성 : 폼의 입력된 값들을 전송받을 서버의 클래스명을 입력한다.
* method 속성 : get/post 방식으로 전송방식을 지정한다.
-> get: 사용자가 입력한 값이 쿼리스트링(URL)에 노출됨. -> post: 사용자가 입력한 값이 쿼리스트링에 노출되지 않음. * target 속성 : 현재의 창을 활용하는지 새 창을 활용하는지를 결정한다.

 

submit 버튼을 눌렀을 때 action에 지정한 경로로 method에 지정한 방식으로 input 태그 값을 전달한다.
